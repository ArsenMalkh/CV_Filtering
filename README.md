# CV_Filtering

# Запуск Программы

 ```bash
 chmod +x run.sh
 ./run.sh
 ```
 ### после этого получаем, исполнаемый файл, который можно запустить
 ``` bash
 # ./Filter <input_image> <output_image> <R> <naive|huang|const|opencv>
 ./Filter input_images/image1.bmp output_images/filtered_image1.bmp 10 const
 
 ```
